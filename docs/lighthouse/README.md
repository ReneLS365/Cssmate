# Lighthouse CI notes

Environment variables used by `tools/check-lighthouse-perfect.mjs`:

- `CSSMATE_LH_PERF_MIN`: Minimum performance score (0-100) required to pass.
- `CSSMATE_LH_PERF_TARGET`: Target performance score used for logging.
- `CSSMATE_LH_LCP_MAX_MS`: Maximum LCP in milliseconds.
- `CSSMATE_LH_CLS_MAX`: Maximum CLS value.

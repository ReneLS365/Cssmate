# Legacy materiale

- `features/pctcalc/` – frakoblet procentsats-beregner.
- `ui/numpad.*` – gamle numpad-stubs som blev erstattet af implementeringen under `js/`.

Indholdet bevares som reference, men deployment og service worker bruger ikke filerne.
